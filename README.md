# AI-Powered-Breast-Cancer-Diagnostic-System-in-Gradio


🧠 Breast Cancer Diagnostic Assistant

An AI-powered web application that predicts whether a breast tumor is Benign (non-cancerous) or Malignant (cancerous) using machine learning techniques and the Wisconsin Breast Cancer dataset.

🚀 Features
🔍 Real-time tumor classification (Benign / Malignant)
🎚️ Interactive sliders for input features
📊 Confidence score and malignancy risk calculation
📈 Visual risk level indicator
🕸️ Feature radar chart for better interpretation
🌐 Simple and user-friendly interface using Gradio
🧪 Input Features

The model uses the following six medical parameters:

Radius Mean
Texture Mean
Perimeter Mean
Area Mean
Smoothness Mean
Compactness Mean
🧠 Machine Learning Model
Dataset: Wisconsin Breast Cancer Dataset
Algorithm: “At present, a heuristic-based weighted scoring approach is used to simulate predictions. However, the architecture is designed to support integration of supervised learning models such as Logistic Regression, SVM, or Random Forest trained on the Wisconsin dataset.”
Libraries Used:
Python
Scikit-learn
Pandas
NumPy
🖥️ Tech Stack
Frontend/UI: Gradio
Backend: Python
ML Libraries: Scikit-learn, Pandas, NumPy
